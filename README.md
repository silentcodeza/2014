## 🚀 Getting Started

### Prerequisites
- **Java JDK 8+** installed
- A Java IDE such as **IntelliJ IDEA**, **Eclipse**, or **NetBeans**
- Or simply the command line with `javac` and `java`

### Running Java Files

```bash
# Clone the repository
git clone https://github.com/silentcodeza/2014.git
cd 2014

# Navigate to any exercise folder
cd 1

# Compile a Java file
javac FileName.java

# Run the compiled class
java FileName
```
